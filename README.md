OCR pipeline for transcription and translation of Greek historical texts using Python and Tesseract.
# Greek OCR Text Transcription Project

This project focuses on extracting and processing Greek text from historical documents using OCR techniques.

Technologies Used
- Python
- Tesseract OCR
- Image preprocessing
- Text processing

Description
The project processes images containing Greek text from three historical periods:
- Ancient Greek
- Greek Minuscule
- Modern Greek

The OCR pipeline extracts and processes text from these documents. Successful transcription and translation were achieved for modern Greek texts while earlier scripts were analyzed for OCR limitations.

Purpose
This project demonstrates OCR pipeline development and text processing techniques for historical language digitization.
